# Guia Pessoal de Barcelona — versão curada

Publicado em: https://hctancredi.github.io/guia-barcelona/

Esta versão transforma as dicas brutas do TikTok em um guia de viagem mais usável:

- ranking por categoria
- nota Hermes de prioridade
- descrição básica de cada lugar
- motivo para considerar
- prato/experiência popular
- nota pública quando verificada parcialmente
- link para Google Maps
- link para TikTok salvo
- fonte pública quando disponível

## Arquivos

- `index.html` — página publicada.
- `barcelona_curated_guide.json` — dataset curado da versão atual.
- `barcelona_guide_data.json` — dataset bruto anterior, mantido para referência.

## Categorias

- Comer & cafés
- Noite & drinks
- Passeios & pontos
- Praias & bate-voltas

## Observação

Quando a página diz “sem nota verificada”, significa que a indicação veio dos vídeos salvos e/ou curadoria turística, mas não encontrei uma fonte pública estável o bastante para afirmar nota agregada nesta rodada.
